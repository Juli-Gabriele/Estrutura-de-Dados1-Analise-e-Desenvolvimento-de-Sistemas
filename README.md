# Lista de Exercícios – Lógica e Algoritmos

Repositório contendo exercícios desenvolvidos na disciplina de Estrutura de Dados I, com foco em lógica de programação e algoritmos.

 Curso: Tecnologia em Análise e Desenvolvimento de Sistemas
 Instituição: Instituto Federal do Triângulo Mineiro (IFTM)
 Período: 3º Período

 # Exercicio 1

Programa que trabalha com um vetor de notas e realiza:
Cálculo da média aritmética
Contagem de notas acima da média
Identificação da maior nota (sem ordenação)
Conceitos aplicados foram vetores,estruturas de repetição,comparações e variáveis acumuladoras.

 # Exercicio 2
 
 Algoritmo que calcula o menor número de cédulas para um valor informado.

 Cédulas utilizadas: R$ 50 R$ 20 R$ 10

 Conceitos aplicados:Divisão inteira , Operador resto (%), Lógica matemática;
 
  # Exercicio 3
Algoritmo que verifica critérios de segurança:

✔ Mínimo de 8 caracteres
✔ Presença de caractere especial

Conceitos aplicados: Strings,Percurso de arrays,Validação lógica.
